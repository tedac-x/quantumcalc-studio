# QuantumCalc: Tactical Computation System

A native Android application wrapper built around an optimized, custom calculation interface. The interface features an immersive, dark-themed design optimized for high-performance visual clarity.

## 🛠️ Project Architecture
* **Native Android Wrapper:** Configured using a clean, full-screen immersive layout with the default action bar removed (`NoActionBar`) to allow the custom UI to occupy the entire screen viewport.
* **Core Web Assets:** The tactical user interface, styling, and computation logic run directly out of the local assets directory located at: `app/src/main/assets/`.
* **Application Icon:** Customized with a 1:1 minimalist design integrated using Android Studio's Image Asset Studio.

## 📚 Technical Specifications & References
* **Target Build Environment:** Android Studio
* **UI Components:** Material Design components configured via the application's native theme styles.
* **Primary Target Branch:** `master` (Contains the complete source code and working assets)
